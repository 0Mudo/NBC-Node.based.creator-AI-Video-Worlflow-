## 0. 先读这里（无门槛说明）

### 0.1 你需要先知道的 6 个词

- **项目**：一个创作“文件夹”。每个项目有自己的画布（节点/连线）、灵感（剧本/分镜/三卡）、时间线、素材关联。
- **节点**：一个功能模块（例如“角色卡”“提示词”“GPT 图像生成”“输出”）。节点之间可以连线。
- **连线**：数据流动的方向。上游节点把文字/图片/视频传给下游节点。
- **Provider**：一组“接口配置”。例如 GPT Image 2、Seedance、OSS、飞书云盘都属于 Provider，需要在“设置面板”里配置密钥和端点。
- **素材（Asset）**：图片/视频/文本卡片等资产，统一管理在“素材浏览器”。
- **时间线**：把多个分镜片段按时间顺序排成视频（还支持字幕、对话、BGM 轨道）。

### 0.2 NBC 能做什么（用一句话）

把“想法 → 剧本 → 分镜 → 角色/场景/物品卡 → 提示词 → 文生图/图生视频 → 素材管理 → 时间线 → 导出视频”串成一套可视化流程。

### 0.3 安全与数据位置

- NBC 是桌面应用（Electron）。绝大多数数据保存在本地（浏览器 localStorage + 本地素材目录）。
- API Key / Secret 会存本地。你可以在“设置 → 导出配置”时选择是否**包含密钥**。
- 生成结果可以同时保存到：本地 / OSS / 飞书（见第 19 章）。

## 1. 产品简介

NBC（Node-Based Creator）是一款**节点式素材创作器**。你通过拖拽节点、连线、点击运行，就能把上游信息自动组合成提示词并驱动下游模型生成。

### 核心能力

| 能力 | 说明 |
|------|------|
| 文生图 | GPT Image 2、Banana Image |
| 文生视频 / 图生视频 | Seedance 2.0（支持首帧/首尾帧/多模态/视频编辑/延长等模式） |
| 灵感生产 | 灵感编辑器：剧本/分镜/三卡 + AI 辅助 + 版本管理 |
| 素材管理 | 本地扫描 + OSS 拉取 + 飞书云盘同步 + 标签/项目归属 |
| 时间线 | 多轨道分镜编排 + 批量生成 + 导出脚本/导出视频 |
| 三路保存 | 输出节点一键同时保存到本地/OSS/飞书 |

## 2. 安装与启动

### 2.1 启动方式（开发/生产）

如果你是开发者：

```bash
npm run dev            # 纯 Web 模式，浏览器访问 http://localhost:5173/
npm run electron:dev   # Electron 桌面应用模式（推荐体验完整功能）
```

生产模式：

```bash
npm run electron:prod  # 构建后启动 Electron 应用
```

### 2.2 首次启动会发生什么

- 第一次启动会弹出“首次配置向导”（可跳过，后面随时能在设置面板里改）。
- 推荐先完成：GPT Image 2 / Seedance / LLM（提示词优化）/ OSS / 飞书云盘 的配置，否则相关功能会提示“未配置”。

| API 设置 | 齿轮图标 | 打开“设置面板”（第 5 章） |
| 提示词库 | 书本图标 | 打开“提示词库”（第 16 章） |
| 视频导出 | 下载图标 | 打开“视频导出”（第 14 章） |
| 生成分析 | 柱状图 | 打开“生成分析”（第 17 章） |

### 3.2 Dock 面板操作（适用于所有面板）

- **拖拽 Tab 标题**：移动面板位置，支持左右/上下分栏。
- **关闭 Tab**：Tab 右侧关闭按钮（部分核心面板可能不可关闭）。
- **滚动**：大部分面板内部支持鼠标滚轮滚动。

## 4. 首次配置向导

首次启动会出现 4~5 步向导（每步可跳过）：

1. 欢迎页：介绍产品。
2. GPT Image 2：配置图像生成 Provider（API Key）。
3. Seedance 2.0：配置视频生成 Provider（火山方舟 API Key）。
4. LLM（提示词优化）：配置提示词优化的模型端点与 API Key。
5. 完成：提示哪些已配置/已跳过。

如果你关闭/跳过了向导，后续请到：**顶部工具栏 → API 设置** 里配置（第 5 章）。

## 5. 设置面板（含 OSS / 飞书云盘配置）

打开方式：顶部工具栏 → **API 设置**（齿轮）。

设置面板分为两块：

- **通用设置**：本地素材目录、操作人用户名、项目自动备份间隔
- **API 配置**：所有 Provider（GPT/Seedance/LLM/OSS/飞书/自定义 Provider）

### 5.1 通用设置（每一项怎么用）

#### 5.1.1 默认本地素材目录 / 输出位置

- 你会看到一行路径（例如 `H:\素材库`），右侧有 **选择目录**。
- 点击 **选择目录**：弹出系统目录选择框，选定后：
  - “素材浏览器 → 扫描本地文件夹”的默认路径会更顺手
  - 输出节点保存到本地时也会以这里为主目录

#### 5.1.2 操作人用户名

- 用途：用于记录生成日志、飞书同步队列的“操作者”字段（便于团队区分是谁生成/同步的）。
- 建议填写：你的昵称或工号（例如 `NBC User`）。

#### 5.1.3 项目自动保存备份

- 用途：定时把项目导出为 `.nbc.json` 文件（类似“自动备份存档”）。
- 下拉选项：关闭 / 每 5 分钟 / 每 10 分钟 / 每 30 分钟 / 每 1 小时。

### 5.2 API 配置（Provider 面板通用操作）

你会看到一张张 Provider 卡片（可折叠），例如：

- GPT Image 2
- Banana Image
- Seedance 2.0
- LLM（提示词优化）
- 阿里云 OSS
- 飞书云盘

每个 Provider 卡片都支持以下按钮/控件：

| 控件/按钮 | 作用 |
|---|---|
| 展开/收起 | 点击卡片标题区域展开/折叠配置区 |
| 启用/禁用（Toggle） | 临时关闭某个 Provider（不删除配置） |
| 显示/隐藏密码（眼睛） | 密码字段可切换明文显示 |
| 测试连接 | 对默认端点发起 GET 请求，判断网络是否可达（部分服务返回 HTTP 错误也会视为“可达”） |
| 保存配置 | 把你输入的内容写入本地配置 |
| 移除 | 删除 Provider（一般不建议删除内置 Provider） |

### 5.3 OSS 配置（阿里云 OSS）

目标：让 NBC 能把生成结果上传到 OSS，并在素材浏览器里拉取 OSS 文件列表。

#### 5.3.1 你需要准备什么

- AccessKey ID
- AccessKey Secret
- Bucket Name（桶名）
- Region（例如 `oss-cn-shenzhen`）
- OSS Endpoint URL（通常类似 `https://oss-cn-shenzhen.aliyuncs.com`）

#### 5.3.2 在 NBC 里怎么填

1. 顶部工具栏 → **API 设置**
2. 找到 Provider：**阿里云 OSS**
3. 点击卡片展开
4. 依次填写：
   - AccessKey ID
   - AccessKey Secret
   - Bucket Name
   - Region
   - 端点 URL（如果你的 OSS Endpoint 不同）
5. 点击 **保存配置**
6. 点击 **测试连接**（可选）

#### 5.3.3 常见填法示例（仅示例）

- Region：`oss-cn-shenzhen`
- 端点 URL：`https://oss-cn-shenzhen.aliyuncs.com`

### 5.4 飞书云盘配置

目标：让 NBC 能从飞书云盘指定文件夹同步素材，并在“素材浏览器”里直接浏览/拖拽使用。

#### 5.4.1 你需要准备什么

- App ID（飞书应用 ID）
- App Secret（飞书应用密钥）
- 文件夹 Token（你要同步的飞书文件夹 token）

#### 5.4.2 在 NBC 里怎么填

1. 顶部工具栏 → **API 设置**
2. 找到 Provider：**飞书云盘**
3. 点击卡片展开
4. 依次填写：
   - App ID
   - App Secret
   - 文件夹 Token
   - 端点 URL（一般保持默认 `https://open.feishu.cn/open-apis`）
5. 点击 **保存配置**
6. 点击 **测试连接**（可选）

#### 5.4.3 配置后如何使用

见第 7 章「飞书云盘同步」：在素材浏览器里点击“飞书云盘”按钮即可拉取文件列表，并缓存到本地素材目录。

### 5.5 配置导入/导出（团队协作常用）

Provider 区域右上角有两个按钮：

- **导入**：从 JSON 文件导入 Provider 配置
- **导出**：导出 Provider 配置（会弹出“导出全部配置”窗口）

导出弹窗里有一个重要复选框：

- **包含 API Key 等密钥**
  - 勾选：导出文件包含密钥（适合自己多台电脑同步配置）
  - 不勾选：导出文件会把密钥字段置空（适合给团队成员共享配置模板）

## 6. 项目管理与工作流库

### 6.1 项目管理面板（Project）

你会在面板顶部看到按钮：

| 按钮 | 用途 |
|---|---|
| 保存当前项目 | 把当前画布节点/连线保存到项目存档 |
| 导出工作流 | 导出 `.nbc.json` 文件（可分享/备份） |
| + 新建项目 | 创建新项目并切换 |

项目列表中每个项目条目支持：

- 单击：切换项目
- 悬停出现 ✏️：重命名（Enter 确认 / Esc 取消）
- 悬停出现 🗑️：删除（不能删除当前活跃项目）

### 6.2 画布工具栏里的“保存/导入/导出”

在节点画布顶部（FlowEditor 工具栏）也有一组按钮：

| 按钮 | 用途 |
|---|---|
| 保存项目 | 保存当前 nodes/edges 到当前项目 |
| 导出文件 | 导出 `.nbc.json` |
| 导入文件 | 导入 `.nbc.json` 并覆盖当前画布 |
| 全部运行 | 运行画布内所有生成节点 |
| 删除 | 删除当前选中的节点/连线 |

### 6.3 工作流库（Workflow Library）

用途：把常用画布保存为“模板”，后续一键套用。

| 按钮/动作 | 用途 |
|---|---|
| 右上角 “+” | 将当前画布保存为工作流（输入名称后 ✓ 保存 / X 取消） |
| 点击某个工作流 | 应用到当前项目（会提示确认，并覆盖当前画布） |
| 悬停 ✏️ | 重命名 |
| 悬停 🗑️ | 删除 |

## 7. 素材浏览器（素材库）

素材浏览器用于统一管理：本地素材、OSS 素材、飞书云盘素材、以及各种文本卡片资产。

### 7.1 顶部按钮与开关

| 按钮/控件 | 用途 |
|---|---|
| 批量选择 | 进入多选模式（素材卡片左上角出现勾选框） |
| 网格/列表视图 | 两种展示方式切换 |
| 扫描本地文件夹 | 导入本地文件夹中的图片/视频 |
| OSS | 从 OSS 拉取远程素材列表 |
| 飞书云盘 | 从飞书云盘指定文件夹同步文件 |
| 全选当前 / 取消全选 | 多选模式下快速选择 |
| 删除 / 取消 | 多选模式下批量删除或退出 |

### 7.2 筛选与搜索

- **按分类**：按标签过滤（角色卡/场景卡/GPT Image/Seedance/输出等）。
- **按项目**：按项目归属过滤（全部/未分配/某个项目）。
- **搜索框**：按素材名称模糊搜索。

### 7.3 素材卡片（单个素材的操作）

| 操作 | 方式 |
|---|---|
| 选中 | 单击卡片 |
| 全屏查看 | 双击缩略图 |
| 拖到画布 | 拖拽到画布会自动创建对应节点（图片/视频 → 素材输入节点；文本卡 → 对应卡片节点） |
| 编辑标签 | 悬停点击 ✏️ |
| 删除素材 | 悬停点击 🗑️（OSS 素材会尝试删除远端） |

### 7.4 本地文件夹扫描

1. 素材浏览器工具栏点击 **扫描本地文件夹**
2. 选择一个目录
3. NBC 会扫描其中的图片/视频并加入素材库

提示：默认目录可在“设置面板 → 通用设置 → 默认本地素材目录”里设置。

### 7.5 OSS 同步（拉取）

1. 确保已完成 OSS Provider 配置（第 5.3）
2. 素材浏览器点击 **OSS** 按钮
3. 远端文件会出现在素材列表中

### 7.6 飞书云盘同步

1. 确保已完成飞书云盘 Provider 配置（第 5.4）
2. 素材浏览器点击 **飞书云盘** 按钮
3. NBC 会列出指定文件夹文件，并把未缓存的文件下载到本地素材目录的飞书缓存子目录

### 7.7 媒体查看器（全屏预览）

双击素材缩略图进入全屏查看：

| 操作 | 快捷键 |
|---|---|
| 上一个/下一个素材 | ← / → |
| 关闭 | Esc |
| 播放/暂停（视频） | Space |
| 切换信息面板 | I |
| 切换全屏（图片） | F |

## 8. 灵感编辑器（剧本/分镜/三卡）

灵感编辑器是“从文字到结构化素材”的核心面板。它有 3 步：

1. 第 1 步：剧本
2. 第 2 步：分镜
3. 第 3 步：三卡（角色卡/场景卡/物品卡）

并且支持：

- 自由文本编辑（默认）
- 结构化编辑（把剧本解析为场次卡、分镜解析为镜头卡、三卡解析为表单）
- AI 对话辅助（右侧聊天）
- 版本历史（记录/恢复/删除）
- 一键提取三卡、一键生成角色形象

### 8.1 顶部“步骤”与“子标签”

| 你会看到 | 作用 |
|---|---|
| 第1步：剧本 / 第2步：分镜 / 第3步：三卡 | 切换当前编辑的内容类别 |
| 角色卡 / 场景卡 / 物品卡 | 仅在“三卡”步骤下出现，用于切换具体卡片类型 |

### 8.2 编辑模式：自由文本 vs 结构化

灵感编辑器顶部会有“模式切换”按钮：

- **自由文本**：直接写一段文字（更像写文档）
- **结构化**：把内容拆成可编辑的结构（更像填表/卡片）

#### 8.2.1 剧本/分镜的结构化按钮

| 按钮 | 用途 |
|---|---|
| 解析为卡片（剧本/分镜） | 把当前文本解析成“场次/镜头”卡片结构 |
| 写回文本（剧本/分镜） | 把结构化内容再拼回一段完整文本写入“当前版” |

#### 8.2.2 三卡的结构化按钮

| 按钮 | 用途 |
|---|---|
| 解析为卡片（三卡） | 把当前文本解析成角色/场景/物品的结构化表单 |
| 写回文本（三卡） | 把结构化表单再导出为文本写入“当前版” |

### 8.3 三卡增强（常用按钮）

| 按钮 | 用途 |
|---|---|
| 一键提取三卡 | 从当前剧本 + 分镜中自动提取角色/场景/物品信息，并入库到素材库 |
| 一键生成角色形象 | 对素材库内的角色卡批量生成角色参考表（PNG），并展示进度 |
| 从素材库选择 | 从素材库选择一条资产加载到当前分类 |
| 保存当前 | 把当前类别内容保存/更新到素材库（通常生成文本卡） |
| 全部保存 | 一次性保存“剧本 + 分镜 + 三卡”到素材库 |

### 8.4 版本与历史（防止写崩）

| 控件 | 用途 |
|---|---|
| 自动记录（下拉） | 设置自动保存版本的间隔（关闭/1/5/10/30 分钟） |
| 手动记录 | 立刻保存一个版本快照 |
| 版本历史 (N) | 打开侧边栏，查看版本列表 |
| 恢复此版本 | 把某个历史版本恢复为当前内容 |
| 删除记录 | 删除某条版本记录 |

### 8.5 角色卡里的“角色形象”区域（生成参考表）

当你在“三卡 → 角色卡”时，会看到和图片相关的按钮：

| 按钮 | 用途 |
|---|---|
| 上传参考图 | 选择本地图片作为参考 |

### 9.5 节点类型逐个详解（每个端口/按钮怎么用）

下面按“你在画布上能看到的节点”逐个说明。

#### 9.5.1 素材输入节点（Asset Input）

用途：把“素材库里的图片/视频”作为下游节点的参考图/参考视频输入。

- **如何创建**：从素材浏览器拖拽一张图/一个视频到画布，会自动生成素材输入节点。
- **输出端口**：右侧一个输出点（供下游连接）。
- **检查器字段**：素材 ID（只读）。

常见连法：

- 素材输入 → GPT 图像生成节点的 `image`
- 素材输入 → Seedance 节点的 `image`（首帧/首尾帧）或 `video`（视频编辑/多模态）

#### 9.5.2 角色卡节点（Character Card）

用途：描述“角色长什么样”，并可携带一致性参数供下游引用。

关键字段（检查器）：

- 名称、外观描述、参考图片路径
- 多角色（逗号分隔 ID）
- 一致性：面部一致性提示词、体型/着装一致性、负面提示词、推荐种子、参考图 URLs
- TTS：音色 ID、语速、音调（用于对白配音扩展）

输出端口：右侧输出点（提供角色文本给下游提示词节点）。

#### 9.5.3 场景卡节点（Scene Card）

用途：描述“在哪儿发生”，例如环境、光照、氛围。

关键字段：

- 名称、场景描述、参考图片路径
- 灵感库：快速选择预设场景

输出端口：右侧输出点。

#### 9.5.4 物品卡节点（Item Card）

用途：描述道具/物品外观（例如“银色冲锋枪”“医疗箱”）。

关键字段：

- 名称、物品描述、参考图片路径
- 灵感库：快速选择预设物品

输出端口：右侧输出点。

#### 9.5.5 剧本/分镜节点（Script）

用途：输入一段剧本或分镜描述文字，作为下游提示词的变量来源。

关键字段：

- 剧本/分镜文本（支持模板变量写法，例如 `{{角色}}`）

输出端口：右侧输出点。

#### 9.5.6 提示词节点（Prompt）

用途：把上游角色/场景/物品/剧本信息组合成最终提示词，并可用变量替换。

端口：

- 左侧：输入端口（接上游卡片/剧本）
- 右侧：输出端口（接生成节点的 `prompt`）

关键字段：

- 提示词文本：支持 `{{character}}` `{{scene}}` 等变量（变量会由执行引擎从上游收集替换）
- AI 提示词优化器：见第 10 章

#### 9.5.7 GPT 图像生成节点（GPT Image 2）

用途：把提示词变成图片。

端口：

- `prompt`（左侧 25%）：接提示词节点
- `image`（左侧 55%）：接素材输入（参考图）
- `output`（右侧）：接输出节点

按钮：

- 右上角 **生成按钮**：图标随状态变化
  - ▶️ 空闲：点击开始生成
  - 🔄 运行中：禁用
  - ✅ 完成：生成成功
  - ❌ 失败：失败（可再次点击重试）

常用字段：

- 模型：`gpt-image-2-vip` / `gpt-image-2`
- 画面比例/分辨率、图片质量
- 参考图 URL（也可以通过连线传入）
- 并发数量（一次生成多张）
- 风格选择（从素材库加载）

#### 9.5.8 Banana 图像生成节点（Banana Image）

用途：另一条文生图通道（同样支持参考图）。

端口：

- `prompt`、`image`、`output`（与 GPT Image 类似）

按钮：

- 右上角“生成”按钮（同样是 ▶️/🔄/✅/❌ 状态）

常用字段：

- bananaAspectRatio（例如 `1024x1024`）
- bananaModel（默认 `gpt-image-2`）

#### 9.5.9 Seedance 视频生成节点（Seedance 2.0）

用途：文本/图片/视频驱动生成视频。

端口：

- `prompt`（左侧 20%）：接提示词
- `image`（左侧 45%）：接素材输入（首帧/参考图）
- `video`（左侧 70%）：接素材输入（参考视频）
- `output`（右侧）：接输出节点

按钮：

- 右上角“运行”按钮：▶️/🔄/✅/❌

关键字段（检查器）：

- 模型：Seedance 2.0 / Seedance 2.0 Fast
- 生成模式：文生视频 / 图生视频·首帧 / 图生视频·首尾帧 / 多模态参考 / 视频编辑 / 视频延长
- 分辨率、宽高比、时长
- 推理模式：在线 / 离线（flex）
- 生成有声视频、返回尾帧图、启用联网搜索
- 并发数量、参考素材数量规划
- 风格选择

#### 9.5.10 输出节点（Output）

用途：定义生成结果怎么保存（本地/OSS/飞书），并提供“打开”入口。

端口：

- 左侧 `input`：接任意生成节点（GPT Image / Banana / Seedance）

关键字段（检查器）：

- 保存到本地（复选框）
- 上传到 OSS（复选框）
- 同步到飞书（复选框）

画布上的按钮：

- **打开**：当节点已有本地路径或 OSS URL 或结果 URL 时可点，打开文件/链接。

## 10. AI 提示词优化

AI 提示词优化器位于：**提示词节点（Prompt）→ 右侧检查器**。

### 10.1 你会看到的按钮

| 按钮 | 用途 |
|---|---|

| 采用 | 把优化后的提示词写回节点 |
| 再试 | 重新优化一次 |
| 放弃 | 关闭优化结果，不修改原文本 |

2. 在提示词节点写一个“粗提示词”（哪怕很短也行）
3. 点击 **AI 优化提示词**

## 20. 完整全流程产出案例（覆盖全部功能）

目标：用 NBC 从 0 产出一个“3 镜头短视频”，并覆盖所有核心功能：设置、模板、灵感、素材、节点生成、提示词优化、队列/日志、时间线、导出、三路保存。

### 20.1 先把环境配齐（只需做一次）

1. 打开 **设置面板**
2. 在 **通用设置**：
   - 选择默认本地素材目录（建议 `H:\素材库` 或你常用的素材盘）
   - 填写操作人用户名（例如 `Alice`）
   - 打开“项目自动保存备份”（例如每 10 分钟）
3. 在 **API 配置**：
   - GPT Image 2：填 API Key，保存，测试连接（可选）
   - Seedance 2.0：填火山方舟 API Key，保存
   - LLM：填端点+API Key+model（用于提示词优化），保存
   - 阿里云 OSS：按第 5.3 填好并保存
   - 飞书云盘：按第 5.4 填好并保存
4. （可选）点击 **导出**，导出一份“不含密钥”的配置给团队成员使用

### 20.2 创建项目并套用模板

1. 打开 **项目管理** 面板，点击 **+ 新建项目**，命名：`巴别塔短片`
2. 顶部工具栏打开 **模板市场**
3. 选择 **完整创作工作流**，点击 **使用**
4. 返回画布，你会看到节点已经摆好并连线

提示：如果你不小心把面板拖乱了，点顶部的 **恢复默认布局**。

### 20.3 用灵感编辑器产出剧本/分镜/三卡，并入库素材

1. 打开 **灵感编辑器**
2. 在 **第1步：剧本** 写入（示例，随便写也行）：

```
蜂医在巴别塔大厅发现自毁装置即将启动，疾风冲入制止。二人争执后决定联手解除。
```

3. 在 **第2步：分镜** 写入（每行一镜头）：

```
全景·巴别塔大厅警报闪烁，蜂医抬头看向倒计时 6秒 淡入
中景·疾风冲入画面「停下！你会毁掉所有人」 5秒 硬切
特写·蜂医手指停在按钮上，冷静转身「那就一起把它关掉」 4秒 叠化
```

4. 在 **第3步：三卡**：
   - 点 **一键提取三卡**（从剧本/分镜自动提取角色/场景/物品）
   - 进入 **角色卡** 子标签，选中蜂医/疾风，补充外观描述（可选）
   - （可选）点击 **生成角色参考表** 为角色生成统一参考图
5. 点击 **全部保存**：把剧本/分镜/三卡写入素材库（后面可以拖到画布）

### 20.4 用素材浏览器整理素材（本地/OSS/飞书）

1. 打开 **素材浏览器**
2. 点击 **扫描本地文件夹**，把你已有的参考图/参考视频导入（可选）
3. 点击 **OSS**：拉取 OSS 文件列表（验证配置是否正确）
4. 点击 **飞书云盘**：同步飞书文件夹（验证配置是否正确）
5. 找到一张参考图（可选），拖到画布生成一个 **素材输入节点**

### 20.5 组装提示词：提示词库 + AI 优化

1. 顶部打开 **提示词库**
2. 选择一个“镜头/光影”模板，点击 **插入**（已复制到剪贴板）
3. 回到画布，选中 **提示词节点**，把剪贴板内容粘进去，并加一句你的需求，例如：

```
在 {{scene}} 中，{{character}} 正在阻止自毁装置启动。电影感，紧张氛围。
（此处粘贴提示词库里的镜头/光影模板）
```

4. 在右侧检查器点击 **AI 优化提示词**
5. 觉得满意就点 **采用**

### 20.6 生成图像与视频，并开启三路保存

1. 选中 **输出节点**，在检查器勾选：
   - 保存到本地 ✅
   - 上传到 OSS ✅
   - 同步到飞书 ✅
2. 在画布上先点击 **GPT 图像生成节点** 的 ▶️，生成关键帧图
3. 生成成功后，再点击 **Seedance 视频生成节点** 的 ▶️，基于提示词+参考图生成视频
4. 输出节点会显示：
   - 本地路径（💾）
   - OSS URL（☁️）
   - 并且有 **打开** 按钮可直接打开文件/链接

### 20.7 生成队列与失败日志排查（一定要会）

1. 打开 **生成队列**
   - 查看任务是否 running/completed/failed
   - 运行中的任务可点“取消”
2. 如果失败：
   - 打开 **失败日志**
   - 复制单条报告或导出全部日志发给开发排查
3. 修复配置后，回到节点点击 ▶️ 重试

### 20.8 把分镜导入时间线并批量生成

1. 打开 **时间线**
2. 点击 **导入分镜**
3. 点击 **从灵感编辑器导入**（直接导入第 2 步分镜）
4. 此时视频轨会出现 3 个片段（空坑位）
5. 点击 **批量生成**：系统会依次触发画布里的生成节点填充空片段
6. 生成完成后，片段状态会变为 done

### 20.9 导出视频（两种方式都走一遍）

方式 A（时间线内置脚本导出）：

1. 时间线点击 **导出视频**
2. 点击 **导出脚本**
3. 下载 `nbc_export_script.txt`，按脚本提示运行 ffmpeg 合并

方式 B（视频导出面板）：

1. 顶部工具栏点击 **视频导出**
2. 选择分辨率、fps、输出路径
3. 点击 **导出**
4. 完成后点 **打开文件夹**

### 20.10 备份与分享（项目/配置）

- 在画布工具栏点击 **导出文件**：得到 `.nbc.json`（项目工作流分享）
- 在设置面板点击 **导出**：得到 `nbc_config_YYYY-MM-DD.json`（Provider 配置分享）

## 22. 常见问题

### 22.1 点击生成提示“未配置 API Key”

- 到：顶部工具栏 → API 设置
- 找到对应 Provider（GPT Image 2 / Seedance / LLM），展开填好 API Key，点保存

### 22.2 OSS/飞书同步按钮点了没反应

- 先确认 Provider 已启用（Toggle 是打开状态）
- 点“测试连接”确认网络可达
- 检查 AccessKey/Secret、AppId/AppSecret 是否正确

### 22.3 导入模板/工作流后画布被覆盖了怎么办

- 模板“使用”会覆盖当前画布，这是设计如此
- 如果你之前导出过 `.nbc.json`，可以再导入回来
- 建议养成习惯：重要改动前先“导出文件”备份

### 22.4 时间线导出没有直接生成 MP4

- 时间线的“导出视频”主要生成 ffmpeg 拼接脚本
- 想直接导出 MP4：用顶部工具栏的“视频导出”面板（Electron 环境下支持）
