## 1. NBC 是什么？

NBC 的全称是 **Node-Based Creator（节点式素材创作器）**。它是一个桌面软件，用来帮助你**用 AI 生成图片和视频**。

打个比方：你可以把 NBC 想象成一块"画布"，在这块画布上你不是直接画画，而是**把一个个功能方块（节点）拖出来、连上线**，组成一条"生成流水线"。然后点击运行，AI 就按照你编排的流程，自动生成你想要的素材。

**NBC 能做什么？**

| 能力 | 说明 |
|------|------|
| 🎨 AI 图片生成 | 通过 GPT Image 2 生成高质量角色立绘、场景图、物品图等 |
| 🎬 AI 视频生成 | 通过 Seedance 2.0 从文字或图片生成短视频 |
| 🖼️ ComfyUI 工作流 | 对接你本地的 ComfyUI，运行自定义工作流 |
| 📝 灵感创作 | 内置灵感编辑器，帮你从剧本一步步过渡到角色卡、场景卡 |
| 🕐 时间线管理 | 像剪辑软件一样管理分镜，每个镜头分配素材 |
| 📦 素材管理 | 集中管理所有生成和历史素材，支持本地/云端 |
| 🔧 自定义配置 | 支持自定义 API 端点、多种 Provider 插件 |

## 2. 第一次启动：初始化配置

第一次打开 NBC 时，你会看到一个**配置向导**（Setup Wizard）。别怕，这个向导只有几页，而且很多步骤可以跳过。

**为什么要配置？**
NBC 本身不内置任何 AI 能力——它需要调用外部的 AI 服务（也就是 API）来生成图片和视频。你需要告诉 NBC：
- 去哪里找 AI 服务（API 地址）
- 你的"通行证"是什么（API Key）

**向导各步骤：**

| 步骤 | 配置什么 | 是否必需 |
|------|----------|----------|
| 欢迎页 | 了解 NBC 是什么 | 直接点"开始设置" |
| GPT Image 2 | 图片生成服务的 API Key | **需要**（不配就无法生成图片） |
| Seedance 2.0 | 视频生成服务的 API Key | 可选（不生成视频可以跳过） |
| AI 提示词优化 | 用于 AI 优化提示词的 API Key | 可选（不用优化功能可以跳过） |
| 完成页 | 确认配置 | 点"开始使用" |

> 💡 **小提示**：如果暂时没有 API Key，可以点击"跳过全部"先进去逛逛。之后随时可以在设置面板里补上。

## 3. 界面总览：各个面板是干什么的？

NBC 的界面采用了**可拖拽拼接的面板布局**。第一次打开时，默认布局如下：

```
┌────────────────────────────────────────────────────────────┐
│  [NBC - Node-based Material Creator]  [模板市场] [主题] [⚙] │ ← 顶部工具栏
├──────────┬────────────────────────────────┬────────────────┤
│  项目管理  │                                │    节点面板     │
│  工作流   │        画布（节点编辑器）          │    [可拖拽]    │
│  素材浏览  │                                │                │
│          │                                ├────────────────┤
│          │                                │    属性检查器    │
│          │                                │    [可拖拽]    │
├──────────┴────────────────────────────────┴────────────────┤
│  时间线  │  生成队列  │  失败日志  │  通知(Agent 动态)        │ ← 底部面板区
└────────────────────────────────────────────────────────────┘
```

下面是每个面板的作用（一句话版）：

| 面板 | 干啥的 |
|------|--------|
| **项目管理** | 创建/切换/重命名/删除项目 |
| **工作流** | 导入/导出 .nbc.json 工作流文件 |
| **素材浏览** | 浏览和管理所有素材（生成的图、本地文件等） |
| **画布（节点编辑器）** | 核心操作区：拖节点、连线、运行 |
| **节点面板** | 列出所有可用的节点类型，点击即可添加到画布 |
| **属性检查器** | 选中一个节点后，在这里修改它的参数 |
| **时间线** | 分镜坑位管理，拖素材填坑 |
| **生成队列** | 看当前有哪些生成任务在排队/运行 |
| **失败日志** | 查看生成失败的原因 |
| **通知面板** | 系统通知：生成成功/失败等 |

> 💡 **小提示**：所有面板都可以**拖拽调整位置和大小**。你觉得哪里不好用，直接拽到舒服的位置就行。

## 4. 项目管理：多项目自由切换

NBC 是多项目的——你可以为不同的创作任务创建不同的项目，每个项目有自己独立的画布节点和素材。

### 4.1 创建项目

1. 点击左侧「项目管理」面板
2. 点击顶部的 **+** 按钮
3. 输入项目名称，按回车

### 4.2 切换项目

在项目列表中，**点击你想要的项目名称**即可切换。切换时 NBC 会自动保存当前项目的状态。

当前活跃的项目会用**高亮颜色**标记，一眼就能分辨。

### 4.3 重命名项目

鼠标悬停在项目上，点击出现的**铅笔图标 ✏️**，直接输入新名字。

### 4.4 删除项目

鼠标悬停在项目上，点击出现的**垃圾桶图标 🗑️**。注意：**正在使用的项目不能删除**，需要先切换到另一个项目。

### 4.5 导入/导出项目

点击画布上方工具栏的按钮：

| 按钮 | 功能 |
|------|------|
| **导出文件** | 把当前项目的所有节点和连线保存为一个 `.nbc.json` 文件 |
| **导入文件** | 从 `.nbc.json` 文件恢复一个项目 |

> 💡 导出的 `.nbc.json` 文件可以发给别人，对方导入后就能看到和你一模一样的工作流。

## 5. 节点编辑器：核心画布

### 5.1 什么是节点和连线？

节点编辑器是 **NBC 的核心**。如果你用过 Blender 的材质节点、Unreal Engine 的蓝图、或是 ComfyUI，你会很熟悉这个概念。如果你没用过也没关系，用一句话概括：

> **节点 = 一个功能方块**，**连线 = 数据从一个方块流向另一个方块**。

每个节点做一件事，你把它们连起来，就组成了一个"工作流"。

打个比方：你想生成一张蜂医（角色）在巴别塔（场景）的全身立绘图。你需要：

```
[角色卡节点] ─┐
              ├──→ [提示词节点] ──→ [GPT图像生成节点] ──→ [输出节点]
[场景卡节点] ─┘
```

- **角色卡节点** 提供了角色的名字和外观描述
- **场景卡节点** 提供了场景的描述
- **提示词节点** 把上面的信息拼成一个完整的 AI 提示词
- **GPT图像生成节点** 调用 AI 生成图片
- **输出节点** 决定了生成好的图片存到哪里

### 5.2 画布操作

| 操作 | 怎么做 |
|------|--------|
| **添加节点** | 在右侧「节点面板」点击某个节点类型（或拖拽到画布上） |
| **移动节点** | 鼠标按住节点，拖动 |
| **缩放画布** | 滚轮上下滚动 |
| **平移画布** | 按住鼠标中键拖动（或按住空白处拖动） |
| **选中节点** | 点击节点（按住 Shift 多选，或框选） |
| **删除节点** | 选中后按 Delete 或 Backspace |
| **创建连线** | 把鼠标移到节点边缘出现的**小圆点**上，按住拖到另一个节点的圆点上 |
| **删除连线** | 点击连线选中，按 Delete |
| **撤销** | Ctrl+Z |
| **重做** | Ctrl+Shift+Z |

> 💡 每个节点左右边缘都有**小圆点（Handle）**。左边的叫"输入"（数据从这里进来），右边的叫"输出"（数据从这里出去）。连线总是从左（输出）往右（输入）连。

### 5.3 所有节点类型逐一讲解

NBC 有 **11 种节点**，分成 4 个大类：

---

#### 📋 卡片类节点（提供描述信息）

**① 角色卡节点（CharacterCard）**
- **作用**：描述一个角色的名字和外观
- **参数**：
  - 角色名（如：蜂医）
  - 外观描述（如：穿白大褂的青年医生，戴金丝眼镜，表情温和……）
  - 参考图链接（可选，提供一张参考图 URL）
  - 灵感库选择（可以从素材库中选一个已有的角色资产）
- **连线**：输出 → 连接到提示词节点或生成节点

**② 场景卡节点（SceneCard）**
- **作用**：描述一个场景/地点
- **参数**：场景名、场景描述、参考图、灵感库选择
- **连线**：同角色卡

**③ 物品卡节点（ItemCard）**
- **作用**：描述一个道具/物品
- **参数**：物品名、物品描述、参考图、灵感库选择
- **连线**：同角色卡

---

#### 📝 文字类节点（提供文本内容）

**④ 剧本节点（Script）**
- **作用**：写一段剧本或分镜文本，给 AI 提供叙事上下文
- **参数**：一大段自由文本
- **连线**：输出 → 连接到提示词节点或生成节点

**⑤ 提示词节点（Prompt）**
- **作用**：编写 AI 生成用的提示词（Prompt），支持模板变量
- **参数**：提示词文本，可以使用 `{{变量名}}` 占位符
- **支持的模板变量**：
  - `{{character}}` → 自动替换为上游角色卡的名字
  - `{{characterAppearance}}` → 自动替换为上游角色卡的外观描述
  - `{{scene}}` → 自动替换为上游场景卡的名字
  - `{{sceneDescription}}` → 自动替换为上游场景卡的描述
  - `{{item}}` → 自动替换为上游物品卡的名字
  - `{{itemDescription}}` → 自动替换为上游物品卡的描述
- **附带功能**：「AI 优化提示词」按钮（见第 10 章）
- **连线**：输入 ← 接收上游节点内容；输出 → 连接到生成节点

> 💡 模板变量的意义：当你改上游角色卡的名字时，提示词会自动更新，不用手动改。

---

#### 🎨 生成类节点（调用 AI）

**⑥ GPT 图像生成节点（GPTImage）**
- **作用**：调用 GPT Image 2 API 生成图片
- **参数**：
  - 模型（gpt-image-2-vip / gpt-image-2）
  - 画面比例（1:1 方形 / 16:9 横屏 / 9:16 竖屏 等 15 种）
  - 画质（自动 / 低 / 中 / 高）
  - 参考图 URL（一行一个）
  - 批量数量（一次生成几张）
- **连线**：输入 prompt ← 接提示词节点；输入 image ← 接参考图来源；输出 → 接输出节点
- **特殊**：点击节点上的 **▶ 运行按钮** 直接开始生成

**⑦ Banana 图像生成节点（Banana）**
- **作用**：另一套图片生成 API（与 GPTImage 类似，不同后端）
- **参数**：模型、比例、参考图、批量数量
- **连线**：同 GPT 图像节点

**⑧ Seedance 视频生成节点（Seedance）**
- **作用**：调用 Seedance 2.0 API 生成视频
- **参数**：
  - 模型（画质版 / 速度版）
  - 生成模式（文生视频 / 图生视频-首帧 / 图生视频-首尾帧 / 多模态 / 视频编辑 / 视频扩展）
  - 分辨率（480p / 720p / 1080p）
  - 画面比例
  - 时长（4-15秒）
  - 生成音频（开关）
  - 返回尾帧（开关，用于连环生成）
  - 联网搜索（开关）
  - 批量数量
  - 参考图/参考视频/参考音频数量
- **连线**：输入 prompt ← 接提示词；输入 image ← 接参考图；输入 video ← 接参考视频；输出 → 接输出节点

**⑨ ComfyUI 节点（ComfyUI）**
- **作用**：对接本地 ComfyUI，运行自定义工作流
- **参数**：工作流名称（对应存储在 ComfyUI 中的 workflow）
- **连线**：输入 prompt ← 接提示词；输入 image ← 接参考图；输出 → 接输出节点

---

#### 📤 输出类节点

**⑩ 输出节点（Output）**
- **作用**：决定生成结果存到哪里
- **参数**（三个复选框，可以多选）：
  - ☑ 保存到本地（`文档\NBC素材` 文件夹）
  - ☑ 上传到 OSS（阿里云对象存储）
  - ☑ 同步到飞书队列
- **连线**：输入 ← 接生成节点的输出

---

#### 📂 素材类节点

**⑪ 素材输入节点（AssetInput）**
- **作用**：引用一个本地/素材库中的图片或视频，作为生成节点的参考图
- **特殊**：这个节点不在节点面板中，只能从素材浏览器**拖拽**到画布上创建
- **连线**：输出 → 接生成节点的 image 输入端

---

### 5.4 从素材浏览器拖素材到画布

NBC 的一个便捷功能是**从素材浏览器直接拖到画布**：

| 拖什么 | 创建什么节点 |
|--------|-------------|
| 拖一张角色卡（text 类型 + Character 标签） | 自动创建**角色卡节点**，并填入名字、描述、参考图 |
| 拖一张场景卡（text 类型 + Scene 标签） | 自动创建**场景卡节点** |
| 拖一张物品卡（text 类型 + Item 标签） | 自动创建**物品卡节点** |
| 拖一张图片/视频 | 自动创建**素材输入节点** |

> 💡 这个功能让你不需要在画布上手动建节点、填参数，直接从素材库一拖就行。

## 6. 灵感编辑器：从剧本到三卡

灵感编辑器是 NBC 的**创意孵化器**。它的设计思路是：帮你从最模糊的"一个想法"开始，通过 AI 辅助，逐步细化成可用的角色卡、场景卡和物品卡。

### 6.1 三步工作流

灵感编辑器分为 **三步（三个标签页）**，建议按顺序使用：

```
第1步：剧本（Script）
  ↓  写一个故事梗概或完整剧本
第2步：分镜（Storyboard）
  ↓  把剧本拆成具体的分镜镜头
第3步：卡片（Cards）
  ↓  从剧本和分镜中提取角色/场景/物品
  ├── 角色卡（Character）
  ├── 场景卡（Scene）
  └── 物品卡（Item）
```

每一步的左侧都会显示**上一步的内容作为参考**（比如你在第2步分镜时，左边可以看到第1步的剧本内容），方便你上下对照。

### 6.2 AI 迭代对话

每一步都有一个**右侧对话面板**，你可以用它和 AI 助手对话：

- 输入你的需求（如："把这个场景写得更紧张一些"）
- AI 会返回修改建议
- 如果 AI 返回的内容包含 `<result>...</result>` 标签，这些内容会自动提取出来
- 点击消息下方的**「采用到当前版本」**按钮，将 AI 的建议应用到编辑器中

> 💡 AI 对话不是一次性的——你可以反复追问、不断修改，像和人聊天一样和 AI 协作。

### 6.3 双版本对比编辑

灵感编辑器的一个特色是**两栏对比编辑**：

| 左栏 | 右栏 |
|------|------|
| **上一版本**（Previous） | **当前版本**（Current） |

- 当 AI 生成新的内容时，它进入右栏（当前版本），旧内容自动移到左栏（上一版本）
- 两栏都可以自由编辑
- 颜色差异标注让你一目了然地看到改了什么

历史版本功能：
- 点击工具栏的 **💾 保存当前版本** 会创建一个版本快照
- 可以设置**自动保存**（每 1/5/10/30 分钟），避免忘保存
- 左侧版本列表显示所有历史版本，可以随时**恢复**或**删除**某个版本

### 6.4 一键提取三卡

当你在第3步（卡片）时，可以点击 **「一键提取三卡」** 按钮。

这个功能会：
1. 把你前面写的剧本 + 分镜全部发给 AI
2. AI 自动分析内容，识别出所有角色、场景、物品
3. 自动创建对应的文本资产，保存到素材库中

> 💡 这个功能特别适合有大量文本需要整理的情况。AI 会帮你从一堆文字中"捞出"关键的角色和场景信息。

### 6.5 批量生成角色参考图

在灵感编辑器的第3步（角色卡标签页），右侧有一个**批量生成角色参考图**的功能：

1. NBC 会列出素材库中所有带 "Character" 标签的文本资产
2. 点击「批量生成」，NBC 会**自动为每个角色生成一张详细的角色参考图**
3. 这些参考图包含全身旋转视角、色板、表情、微表情、服装细节、手势参考等
4. 生成完成后，参考图会自动绑定到对应的角色卡上

> 💡 这是 "节约时间" 神器——你不用一个一个手动给角色生成参考图，点一下全部搞定。

## 7. 素材浏览器：你的素材中心

素材浏览器是你的**内容仓库**。你在 NBC 中生成的所有图片、视频、文本卡都在这里集中管理。

### 7.1 浏览模式

素材浏览器支持两种浏览方式，通过顶部的切换按钮切换：

**① 按分类浏览（By Category）**

按素材的类型标签分组：

| 分类 | 包含什么 |
|------|----------|
| 全部（All） | 所有素材 |
| GPT Image | GPT 图像生成节点产生的图 |
| Seedance | Seedance 视频节点产生的视频 |
| ComfyUI | ComfyUI 节点产生的图 |
| ZzzMap | 全景图 |
| Character | 所有角色卡和角色图 |
| Scene | 所有场景卡和场景图 |
| Item | 所有物品卡和物品图 |
| Output | 输出节点保存的素材 |

**② 按项目浏览（By Project）**

按素材所属的项目分组：
- 当前项目
- 其他项目（按项目名分组）
- 未分类（没有关联任何项目的素材）

> 💡 按项目浏览可以帮助你在多项目工作时快速找到属于某个项目的素材，不会混乱。

### 7.2 搜索功能

素材浏览器顶部的搜索框支持**按名称搜索**（不区分大小写），输入关键字即可过滤。

### 7.3 视图切换

点击 **网格图标** 切换为缩略图网格视图，点击 **列表图标** 切换为详细信息列表视图。

在列表视图中，你可以看到每个素材的名称、类型标签和创建时间，还可以直接**拖拽**到画布上创建节点。

### 7.4 扫描本地文件夹

点击 **「扫描本地文件夹」** 按钮，选择一个本地文件夹，NBC 会扫描其中的图片和视频文件并导入到素材库中。

默认扫描路径为 `H:\素材库`。

在 Electron（桌面应用）模式下，NBC 启动时会自动扫描默认路径。

### 7.5 从 OSS 云端加载

如果你配置了阿里云 OSS（在设置面板中），可以点击 **「从OSS加载」** 按钮拉取云端素材列表。

### 7.6 从飞书云盘加载

如果你配置了飞书云盘凭据，可以点击 **「从飞书加载」** 拉取飞书云盘中的素材。

### 7.7 把素材拖到画布上

参见 [5.4 节](#54-从素材浏览器拖素材到画布)——从素材浏览器拖到画布是创建节点的最快方式。

## 8. 时间线系统：组织分镜

时间线是 NBC 的**分镜管理工具**。你可以把它理解成一个简化版的剪辑分镜板。

### 8.1 时间线的基本概念

- **坑位（Slot）** = 一个镜头/分镜
- **素材池** = 生成好的图片和视频的集合
- **填充** = 把一个素材拖到某个坑位上，表示这个镜头用这个素材

时间线顶部显示：
- 🟢 全部填满 / 🔴 还有空坑的计数
- 总时长

### 8.2 从灵感编辑器导入分镜

最常用的操作：点击「从灵感编辑器导入」按钮，会把你在灵感编辑器第2步中写的分镜文本**自动解析成时间线坑位**。

解析规则很智能，会识别文本中的：
- 镜头类型（全景 / 中景 / 近景 / 特写 / 远景 / 大远景 / 中近景 / 大特写）
- 时长（如 "8秒"、"5s"）
- 转场（硬切 / 淡入淡出 / 叠化 / 擦除 / 滑入 / 缩放）
- 角色名和场景名（自动匹配预设数据）

例如输入文本：
```
全景·蜂医从巴别塔高处跃下 8秒 淡入淡出
```
会自动创建一个坑位：镜头类型="全景"，角色="蜂医"，场景="巴别塔"，时长=8秒，转场="淡入淡出"。

### 8.3 手动创建/编辑坑位

你也可以手动操作：
- 点击 **+** 按钮快速添加一个坑位
- 输入标签、时长（秒），点击「添加」
- 点击坑位上的 ✕ 删除
- 拖拽坑位的拖拽手柄（⠿）可以**调整顺序**
- 点击两个坑位之间的转场标记可以**切换转场类型**

### 8.4 素材池与拖拽填坑

每次你在节点编辑器中运行生成节点，生成的图片/视频会**自动进入素材池**（时间线底部的缩略图区）。

把素材池中的缩略图**直接拖到时间线坑位上**，就完成了填充。

- 空坑 = 红色虚线边框
- 已填充 = 绿色边框 + ✓ 标记
- 鼠标悬停在素材缩略图上会**自动播放视频预览**

### 8.5 调整镜头参数

每个坑位底部可以设置：
- **镜头类型**：8种（全景/中景/近景/特写/远景/大远景/中近景/大特写）
- **时长**：1-60秒
- **转场类型**：6种

## 9. 执行生成：让 AI 开始工作

### 9.1 运行单个节点

每个生成类节点（GPT图像、Seedance、ComfyUI、Banana）上都有一个 **▶ 运行按钮**。

点击运行按钮后：
1. NBC 会自动沿着连线向**上游**查找所有连接的节点
2. 收集上游的角色卡、场景卡、物品卡、剧本、提示词信息
3. **自动替换**提示词中的 `{{变量}}`
4. 调用对应的 AI API 开始生成
5. 生成结果自动显示在节点下方
6. 成功生成的素材**自动加入时间线素材池**
7. 连接了输出节点的话，**自动保存**到配置的位置

节点的状态图标：
| 图标 | 含义 |
|------|------|
| ▶️ | 待运行 |
| 🔄 | 运行中 |
| ✅ | 成功 |
| ❌ | 失败 |

### 9.2 一键运行全部

点击画布顶部工具栏的 **「全部运行」** 按钮，NBC 会**按拓扑顺序**依次运行画布上所有的生成节点。

> 💡 拓扑顺序的意思是：NBC 会先运行被别的节点依赖的节点，确保数据流向正确。你不需要手动排序。

运行完成后，如果有时间线坑位还没填满，NBC 会提醒你。

### 9.3 生成队列怎么看？

底部面板的「生成队列」标签页显示所有正在进行的生成任务：

| 状态 | 显示 |
|------|------|
| 排队中 | ⏰ 时钟图标 |
| 运行中 | 🔄 旋转加载 + 进度条 |
| 已完成 | ✅ 绿色对勾 + 缩略图 |
| 失败 | ❌ 红色叉号 + 错误信息 |

- 鼠标悬停在运行中的任务上，可以点击**取消按钮**终止它
- 点击「清除已完成」一键清空已完成的任务

### 9.4 失败日志：出错了怎么办？

底部面板的「失败日志」标签页记录了所有生成失败的信息：

每个失败记录包含：
- 失败类型（GPT图像 / Seedance / ComfyUI）
- 发生时间
- 是哪个节点
- 具体的错误信息
- 点击可展开查看完整错误详情

你可以：
- 📋 单条复制到剪贴板
- 📤 「导出全部」将所有失败日志导出为文本
- 🗑️ 一键清除所有日志

> 💡 失败日志持久保存在本地磁盘上，即使关了 NBC 再打开，历史失败记录还在。

## 10. AI 提示词优化

在「提示词节点」的属性检查器中，有一个 **「AI 优化提示词」** 按钮。

点击后，NBC 会：
1. 读取你提示词节点的当前内容
2. **自动收集上游连接的所有节点内容**（角色卡、场景卡、物品卡、剧本）
3. 把所有这些信息一起发送给 LLM（大语言模型）
4. LLM 返回优化后的提示词
5. 在属性检查器中显示优化结果

你可以：
- **采用**：把优化结果写入提示词节点
- **重试**：让 AI 再优化一次
- **放弃**：关掉优化结果

> 💡 这个功能的好处是 AI **看到了完整的上下文**，优化出的提示词会更准确、更符合你想要的效果。

## 15. 完整实操案例：从零到生成一张图

让我们用一个完整的例子，走一遍 NBC 的完整工作流。

**目标：生成一张「蜂医从巴别塔高处跃下」的图片。**

---

### 步骤1：创建项目

1. 打开 NBC
2. 左侧「项目管理」面板，点击 **+**
3. 输入项目名：`蜂医-巴别塔`
4. 按回车

---

### 步骤2：添加节点

在右侧「节点面板」中，依次点击添加以下节点：
1. 在「卡片」分类中，点击**角色卡** → 画布上出现角色卡节点
2. 再点一次**角色卡**（我们加两个角色卡，一个蜂医，一个敌人）
3. 在「卡片」分类中，点击**场景卡**
4. 在「文字」分类中，点击**提示词**
5. 在「生成」分类中，点击**GPT图像生成**
6. 在「输出」分类中，点击**输出**

现在画布上有 6 个节点。

---

### 步骤3：配置角色卡

点击第一个角色卡节点，右侧「属性检查器」显示参数：

1. **灵感库下拉**：展开「📋 预设模板」，选择「蜂医」→ 名字和外观自动填入
2. 把角色的**标签**改为「蜂医」

点击第二个角色卡节点：
1. 在「灵感库」中选择另一个角色（比如「老太」或手动输入）
2. 把标签改为「敌人」

> 💡 你也可以直接在「角色名」和「外观描述」输入框中手动填写，不一定要用预设模板。

---

### 步骤4：配置场景卡

点击场景卡节点：
1. 在「灵感库」中选择「巴别塔」→ 场景名和描述自动填入
2. 把标签改为「巴别塔」

---

### 步骤5：配置提示词

点击提示词节点，在提示词文本框中输入：

```
生成一张{{character}}的角色全身立绘，站立姿势，表情坚毅。
{{character}}的外观是：{{characterAppearance}}
场景在{{scene}}，场景描述是：{{sceneDescription}}
画面比例16:9，电影感光影，高质量，细节丰富。
```

> 💡 `{{character}}` 等模板变量会在运行时自动替换为上游角色卡/场景卡的实际内容。

---

### 步骤6：连线

现在把所有节点连起来：
1. 「蜂医」角色卡的 **输出圆点** → 拖到提示词节点的 **输入圆点**
2. 「场景卡」的输出圆点 → 拖到提示词节点的输入圆点
3. 提示词节点的输出圆点 → 拖到 GPT图像节点的 **prompt 圆点**
4. GPT图像节点的输出圆点 → 拖到输出节点的 **输入圆点**

---

### 步骤7：配置 GPT 图像节点

点击 GPT 图像节点：
1. **画面比例**选「16:9」
2. **画质**选「高」
3. **批量数量**设为 1

---

### 步骤8：配置输出节点

点击输出节点：
1. 勾选 ☑ **保存到本地**

---

### 步骤9：运行生成

点击 GPT 图像节点上的 **▶ 运行按钮**。

你会看到：
- 节点状态变成 🔄（运行中）
- 底部「生成队列」中出现一条任务
- 过一会儿状态变成 ✅（成功）
- 节点下方出现生成的图片缩略图
- 图片同时出现在「时间线」的素材池中

---

### 🎉 完成！

你刚刚成功用 NBC 生成了一张 AI 图片。整个过程建立了一条可复用的工作流——下次只需要改角色或场景，再点运行就行了。

## 16. 常见问题 FAQ

### Q：为什么点了运行没有反应？

A：检查以下几点：
1. 生成类节点是否连接了提示词节点？没有输入的生成节点无法运行。
2. API Key 是否配置正确？（设置面板 → GPT Image 2 Provider → 测试连接）
3. 查看「失败日志」面板，里面有具体的错误信息。

### Q：生成的图去哪里了？

A：取决于你的输出节点设置：
- 勾选了「保存到本地」→ 在 `文档\NBC素材` 文件夹
- 勾选了「上传到OSS」→ 在阿里云 OSS bucket 中
- 没有连输出节点 → 图片只在节点上显示，没有持久化保存
- 另外，所有生成的素材都会出现在「素材浏览器」中。

### Q：怎么把现有的素材用到新项目中？

A：在素材浏览器中找到想要的素材，直接**拖到画布上**。根据素材类型会自动创建角色卡/场景卡/物品卡/素材输入节点。

### Q：灵感编辑器的内容会丢吗？

A：不会。灵感编辑器的内容通过 `localStorage` 持久化存储，按项目隔离。只要不清除浏览器数据/NBC 数据，内容就不会丢。

### Q：模板变量为什么没有被替换？

A：模板变量 `{{xxx}}` 只能被**直接连接在上游**的对应节点替换。比如 `{{character}}` 需要提示词节点直接连接到一个角色卡节点。如果中间隔了别的节点可能替换不到。

### Q：可以不连输出节点就生成吗？

A：可以。输出节点只是决定了生成结果是否自动保存。不连输出节点，图片依然会生成并显示在生成节点上，只是需要你手动保存。

### Q：批量生成几张图同时进行？

A：在生成节点的属性中设置「批量数量」即可。比如设为 4，点击一次运行就会同时生成 4 张图。

### Q：Seedance 视频生成要多久？

A：取决于你选择的模式和时长。速度版（fast）比画质版快很多，时长越短越快。生成过程中可以在队列面板看到进度。
